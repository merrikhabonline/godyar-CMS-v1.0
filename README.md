# Godyar CMS v1.09 (Clean GitHub Release)

This repository is a cleaned, deployment-ready version prepared for GitHub.

## Quick start
- See `docs/INSTALL.md` for installation.
- See `docs/DEPLOY_SHARED_HOSTING.md` for cPanel notes.

## Security notes
- Do **not** commit `.env`, `.user.ini`, or `includes/env_path.php`.
- Keep secrets outside the web root.

## Legacy + Modern routing
The app supports language prefixes (`/ar`, `/en`, `/fr`) and legacy includes via a compatibility layer.
