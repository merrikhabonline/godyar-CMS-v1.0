# تثبيت Godyar CMS (نسخة خام ونظيفة)

## قبل البدء
1) أنشئ قاعدة بيانات MySQL على الاستضافة.
2) ارفع ملفات المشروع إلى `public_html` (أو مجلد الموقع).
3) تأكد أن المجلدات التالية قابلة للكتابة:
- `storage/`
- ملف `.env` (سيتم إنشاؤه تلقائياً)

## التثبيت
افتح في المتصفح:
`/install/`

واتبع الخطوات لإنشاء ملف `.env` وبناء الجداول وإنشاء حساب المدير.

## بعد التثبيت (إجباري)
- احذف مجلد `install/` بالكامل.
- احتفظ بـ `.env` ولا تشاركه.

## ملاحظات
- هذه الحزمة نظيفة (بدون كاش/سجلات/ملفات رفع قديمة).
- تم الإبقاء على الكود كما هو لضمان التوافق، مع إزالة النسخ/المجلدات القديمة الواضحة.


### ملاحظة (إعادة تشغيل التثبيت)
- تم جعل تنفيذ SQL متسامحًا مع الأخطاء الآمنة مثل: Duplicate column (1060) و Table already exists (1050)، حتى ينجح التثبيت حتى لو تمت إعادة تشغيل الخطوة 3 أو بقيت بقايا من محاولة سابقة.
- يتم تعطيل FOREIGN_KEY_CHECKS أثناء إنشاء الجداول لضمان حذف الجداول القديمة حتى لو كانت تحتوي علاقات سابقة.
