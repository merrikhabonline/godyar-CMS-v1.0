# Godyar CMS (RAW Clean Package)

## Installation (Shared Hosting)
1. Upload and extract the package into your site root (e.g., public_html).
2. Create a new MySQL/MariaDB database and a user with full privileges on that database.
3. Open: `/install/` in your browser and follow the steps.
4. After success, delete the `/install` folder or keep it (it becomes locked automatically).

## Notes
- This RAW package ships without demo content or cached files.
- Foreign keys are intentionally omitted for maximum compatibility on shared hosting.
